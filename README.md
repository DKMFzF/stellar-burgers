## Дока
Проект на React для заказов межгалактических бургеров, он больше учебные, чем что то продовое, по этому вот тех стек: React, rtt как state-менеджер

## Важно:

Для корректной работы запросов к серверу необходимо добавить переменную BURGER_API_URL в окружение. Сама ссылка находится в файле `.env.example`.
